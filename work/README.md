# 复杂版 Python 计算器

这是一个升级后的命令行 Python 计算器项目，相比基础版，增加了菜单系统、表达式计算、历史记录、更多运算符，以及基础单元测试。

## 功能特性

### 1. 标准计算模式
支持输入两个数字和一个运算符进行计算。

支持的运算符：
- `+` 加法
- `-` 减法
- `*` 乘法
- `/` 除法
- `%` 取余
- `//` 整除
- `**` 幂运算

### 2. 表达式计算模式
支持直接输入表达式，例如：

```text
(2 + 3) * 4
```

支持：
- 数字
- 括号
- 基础算术表达式
- 安全 AST 解析，避免直接执行危险代码

### 3. 历史记录功能
- 自动将计算结果保存到：`work/history.txt`
- 支持查看历史记录
- 支持清空历史记录

### 4. 帮助菜单
内置帮助说明，便于快速查看支持的模式与运算符。

### 5. 输入与异常处理
支持以下异常处理：
- 输入为空
- 非法数字输入
- 非法运算符
- 非法表达式
- 除数为 `0`
- 无效菜单选择

```text
work/

├── test_calculator.py
├── README.md
└── history.txt   # 运行后自动生成或更新
```

## 运行方式

### 启动计算器
在项目主目录执行：

```bash
python3 work/calculator.py
```

### 运行测试
在 `work` 目录下执行：

```bash
cd work
python3 -m unittest test_calculator.py
```
